## 주요 기능 상세

### 1. RAG 기반 AI 챗봇
```
사용자 질문 → 문서 검색 (pgvector) → Gemini API → 답변 생성
- 정책 Q&A
- 판세 분석 상담
- 전략 컨설팅
```

### 2. 판세 감지 AI (Election Radar)
```
뉴스 + 여론조사 → 감정 분석 → 추세 감지 → 경보 시스템
- 실시간 판세 트렌드
- 위험 신호 조기 경보
- 1주일/1개월 예측
```

### 3. 경쟁자 분석
```
경쟁자 데이터 수집 → 강점/약점 분석 → 대응 전략
- 지지율 비교
- 정책 차별성
- 언론 전략 분석
```

### 4. 캠프 코파일럿
```
현재 판세 → 목표 설정 → 최적화 전략 → 액션 플랜
- 주간/월간 계획
- 우선순위 조정
- 성과 예측
```

### 5. 콘텐츠 자동 생성
```
정책 + 판세 분석 → 콘텐츠 작성 → 채널별 최적화
- SNS 포스트
- 뉴스레터
- 보도자료
```

### 6. 후보 개인 코칭
```
후보 성과 → 강점/약점 진단 → 맞춤형 피드백
- 언론 대면 훈련
- 이슈별 대응 가이드
- 연설 분석
```
